# Cromite RPM

RPM packages for Cromite (a Bromite fork with built-in ad blocking and privacy enhancements).
Binary repackaging of the upstream project: https://github.com/uazo/cromite
Based on work done by ayatsuno: https://github.com/ayatsuno/cromite-rpm

## COPR Repository

Ready-to-use RPMs are available in Fedora COPR:
https://copr.fedorainfracloud.org/coprs/atsitimolan/cromite/
